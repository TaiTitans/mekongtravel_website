# mekongtravel_website
 Full project MekongTravel Website
 ```
Mở rộng tính năng và bổ sung AI.
- Sử dụng định vị trí 
- Quản lý và đặt tour cho user
- Tìm địa điểm và mở rộng database(cách chế biến món ăn, lịch sử của địa điểm).
- Bổ sung cho admin quản lý và update địa điểm và ẩm thực.
- Bổ sung AI ( chatbot, tìm kiếm bằng hình ảnh).
- Tối ưu hóa database bằng AWS và Cloudinary cho website đồng bộ hoàn toàn bằng Cloud.
```

```
Công nghệ sử dụng:
- Frontend: Figma, HTML,CSS, JS(VueJS)...
- Backend: Java Spring Boot + API
- Database: MySQL, Cloudinary
```
